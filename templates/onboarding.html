Add New Host

Connect Your Host VM

Follow these steps to connect a new host VM to StagDB. This process will set up your host for creating databases with instant branching capabilities using ZFS.

⚠️ Prerequisites

  • Docker must be installed and running on the host
  • SSH access to the host VM (for remote hosts)
  • ZFS installed (recommended) or sufficient disk space for image/directory storage
  • Available disks, existing ZFS pools, or 20GB+ free space
  • Storage configuration completed before host setup
1

Provide Host Details

Enter the IP address, username, and SSH credentials for your host VM.

2

Verify Prerequisites

StagDB will connect to your host and verify that ZFS and Docker are installed.

3

Configure Storage

Set up storage configuration: use existing ZFS pools, dedicate disks, or create image-based storage.

4

Complete Setup

Review your configuration and complete the host setup process.

Important: Data on the selected ZFS pool may be deleted during the setup process. Make sure you have backups of any important data before proceeding.
